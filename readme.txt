== Description ==

Add a customizable widget to any wordpress theme so your visitors can search the booking.com index with your Affiliate ID on.



== Installation ==

1) Upload to your blog, Activate it.
2) Go to the settings page and fill out the details.
3) Go to the "Appearance > Widgets" page and add the widget wherever you want.

A couple of hours later, check your traffic & bookings via the Booking.com admin panel !
